# Content Writer

Generate marketing content optimized for search and engagement.

## When to Use

- "Write a blog post about X"
- "Create content for this keyword"
- "Draft social posts for this article"
- "Write meta description for this page"
- "Generate content brief"
- "Repurpose this content"

## Content Types

### 1. Blog Post

When asked to write a blog post:

**Gather first:**
- Target keyword (from Search Console opportunities or user input)
- Search intent (informational, transactional, navigational)
- Competitor content (what's ranking?)
- Word count target (typically 1,500-2,500 for SEO)

**Structure:**
```markdown
# [Title with primary keyword near front]

[Hook - 2-3 sentences that address the reader's problem/question]

[Brief overview of what they'll learn - builds trust]

## [H2: First major section]

[Content - aim for 200-400 words per section]

### [H3: Subsection if needed]

[Supporting content]

## [H2: Second major section]

[Content]

## [H2: Practical/Actionable section]

[Steps, tips, or how-to content]

## [H2: Conclusion or Key Takeaways]

[Summarize main points]

[Call to action]
```

**SEO Checklist:**
- [ ] Primary keyword in title (first 60 chars)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Related keywords naturally throughout
- [ ] Meta description (150-160 chars) with keyword
- [ ] Internal links to related content (2-3)
- [ ] External links to authoritative sources (1-2)
- [ ] Images with descriptive alt text

### 2. Meta Description

When asked to write meta descriptions:

**Format:** 150-160 characters, includes:
- Primary keyword (naturally)
- Value proposition or benefit
- Call to action or hook

**Examples:**
```
Learn how to [achieve outcome] with our step-by-step guide. [Benefit statement]. Get started today.

[Primary keyword]: Everything you need to know about [topic]. [Unique angle]. Read our complete guide.

Struggling with [problem]? Discover [number] proven [solutions] that [benefit]. [CTA].
```

### 3. Social Media Posts

When asked to create social content:

**LinkedIn (professional, longer form):**
```
[Hook line - question, stat, or bold statement]

[2-3 sentences expanding on the hook]

[Key insight or takeaway]

[Call to action - comment, share, link]

#relevant #hashtags (3-5 max)
```

**Twitter/X (concise, punchy):**
```
[Core message in <200 chars]

[Link if applicable]
```

**Thread format:**
```
1/ [Hook - the big idea]

2/ [Context or problem]

3/ [Insight 1]

4/ [Insight 2]

5/ [Insight 3]

6/ [Conclusion + CTA]
```

### 4. Content Brief

When asked to create a brief for writers:

```markdown
# Content Brief: [Topic]

## Target Keyword
- Primary: [keyword]
- Secondary: [keyword 2], [keyword 3]
- Search volume: [if known]
- Current ranking: [if known]

## Search Intent
[What is the searcher trying to accomplish?]

## Target Audience
[Who is this for? What do they know? What do they need?]

## Content Goals
- [ ] Rank for [keyword]
- [ ] Drive traffic to [page/product]
- [ ] Generate [leads/signups/sales]

## Competitive Analysis
Top 3 ranking articles:
1. [URL] - [word count] - [key angle]
2. [URL] - [word count] - [key angle]
3. [URL] - [word count] - [key angle]

## Content Gap
What's missing from existing content that we can provide?

## Outline
[H2 and H3 structure]

## Requirements
- Word count: [X]
- Tone: [professional/casual/technical]
- Internal links: [list pages to link to]
- CTA: [what action should reader take]

## SEO Requirements
- Include keyword in: title, H1, first 100 words, 1+ H2
- Meta description: [draft or guidelines]
- URL slug: [suggested]
```

### 5. Content Repurposing

When asked to repurpose existing content:

**Blog → Social:**
- Extract 3-5 key insights as individual posts
- Turn statistics into shareable graphics (provide copy)
- Create a thread summarizing the article
- Pull quotes for LinkedIn posts

**Blog → Email:**
- Condense to 3 main takeaways
- Add personal angle/story
- Strong CTA at end

**Blog → Video Script:**
- Hook (first 5 seconds)
- Problem/context (30 seconds)
- Main points (2-3 minutes)
- CTA (15 seconds)

## Writing Guidelines

### Voice & Tone
- Default to clear, direct language
- Avoid jargon unless audience expects it
- Use "you" and "your" - speak to the reader
- Active voice over passive
- Short paragraphs (2-4 sentences)

### SEO Writing
- Front-load important keywords in headings
- Use keyword variations naturally
- Answer the search query directly and early
- Structure for featured snippets (lists, definitions, tables)
- Include semantically related terms

### Engagement Tactics
- Start with a hook (question, stat, story, bold claim)
- Use subheadings every 200-300 words
- Include bullet points and numbered lists
- Add examples and specifics
- End with clear next step

## Workflow: Keyword → Content

1. **Get keyword data** (from Search Console skill)
   - Current rankings
   - Impressions and CTR
   - Related queries

2. **Analyze intent**
   - What does the searcher want?
   - Are they ready to buy or just learning?

3. **Check competitors**
   - What's ranking?
   - What angle can we take?

4. **Create brief**
   - Outline structure
   - Define requirements

5. **Write content**
   - Follow SEO checklist
   - Match intent

6. **Optimize**
   - Meta description
   - Internal links
   - Images/media

7. **Publish and track**
   - Monitor rankings (Search Console)
   - Track engagement (GA4)

## Notes

- Always ask for target keyword if not provided
- Check Search Console for existing rankings before creating new content
- Prioritize updating existing content over creating new if topic already covered
- Minimum viable blog post: 800 words for most topics, 1,500+ for competitive keywords
